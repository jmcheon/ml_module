### Objective

Broaden the comprehension of the notion of hypothesis. Create a function that takes a vector x of dimension m and an integer n as input, and returns a matrix of dimensions (m√ón). Each column of the matrix contains x raised to the power of j, for j = 1, 2, ..., n:<br/>
&ensp;&ensp;&ensp;&ensp; $x$ \| $x^2$ \| $x^3$ \| $\dots$ \| $x^n$

Such a matrix is called a **Vandermonde matrix**.
